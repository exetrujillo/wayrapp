Quick Start

Authentication Flow

1. Register a new user
curl -X POST https://api.wayrapp.com/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "SecurePass123!",
    "username": "learner123"
  }'
2. Login to get tokens
curl -X POST https://api.wayrapp.com/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "SecurePass123!"
  }'
3. Use access token for authenticated requests
curl -X GET https://api.wayrapp.com/api/v1/users/profile \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
